Cyber Web Scanner adalah alat untuk memeriksa tingkat keamanan dasar sebuah website. Sistem ini menganalisis konfigurasi security headers, mendeteksi konten campuran (mixed content), serta membaca file publik seperti robots.txt. Tujuannya adalah membantu pemilik situs mengenali potensi celah keamanan sejak dini.

2Ô∏è‚É£ Langkah-langkah Menggunakan Website

  1. Buka halaman utama Cyber Web Scanner.
  2. Masukkan alamat situs yang ingin dipindai (contoh: https://example.com).
  3. Klik tombol Pindai Sekarang.
  4. Tunggu beberapa detik hingga hasil pemindaian muncul di layar.
  5. Gunakan menu Riwayat Pemindaian untuk melihat hasil sebelumnya.

3Ô∏è‚É£ Penjelasan Hasil Scan

  • Security Headers ‚Üí Menampilkan header keamanan seperti:
    • Strict-Transport-Security: Memaksa HTTPS agar koneksi aman.
    • X-Frame-Options: Mencegah serangan clickjacking.
    • X-Content-Type-Options: Menghindari sniffing tipe MIME.
    • Content-Security-Policy: Melindungi dari XSS dan injeksi script.
  • Mixed Content ‚Üí Terjadi jika halaman HTTPS memuat elemen HTTP (tidak aman).
  • robots.txt ‚Üí File publik yang memberitahu mesin pencari direktori mana yang boleh atau tidak boleh diakses.
  • Skor & Grade ‚Üí Nilai antara 0‚Äì100 dan huruf A‚ÄìF yang menunjukkan tingkat keamanan.

5Ô∏è‚É£ FAQ (Pertanyaan Umum)

  • Apakah alat ini berbahaya?
    Tidak. Cyber Web Scanner hanya membaca informasi publik situs, tidak melakukan eksploitasi.
  • Boleh digunakan untuk website orang lain?
    Tidak disarankan. Gunakan hanya pada situs yang kamu miliki atau mendapatkan izin dari pemiliknya.
  • Apakah hasil pemindaian disimpan?
    Ya, hasil pemindaian disimpan di database lokal agar kamu dapat melihat riwayat pemindaian sebelumnya.
  • Apakah sistem ini otomatis memperbaiki celah keamanan?
    Tidak. Cyber Web Scanner hanya mendeteksi dan memberi rekomendasi, bukan memperbaiki.
